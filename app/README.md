# Shams ERP UI (Flutter)

Modern ERP UI shell built with Material 3 and glassmorphism.

## Run

```bash
flutter pub get
flutter run -d chrome
```

```bash
flutter run -d windows
```

```bash
flutter run -d android
```

## Build Web

```bash
flutter build web --release --base-href /app/
```

## Structure
- lib/app_shell/
- lib/navigation/
- lib/theme/
- lib/shared/widgets/
- lib/features/
